# HDSnippets

- Configure database.php first

* php artisan migrate
* php artisan migrate --package=cartalyst/sentry
* php artisan db:seed

## Use Cases

- Visitors
	View Snippets
	Register an Account
	Login

- Users
	Write Snippets
	Manage his snipps
	View/Change profile, password
	Vote snipps
	Rate/Comment Snipps
	Logout
